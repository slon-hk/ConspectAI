2. Какие данные мы собираем

2.1. При регистрации

  • Email-адрес — для входа и связи по техническим вопросам
  • Имя пользователя — отображается в личном кабинете
  • Пароль — хранится только в виде криптографического хеша (bcrypt), исходный пароль не сохраняется и нам недоступен

2.2. При использовании сервиса

  • Тексты ваших запросов и ответов ИИ — для обеспечения работы чата, истории диалогов и автоматической генерации карты тем
  • Загружаемые вами файлы (изображения, документы) — для обработки в рамках оказания услуги
  • Метаданные использования — количество отправленных сообщений, использованные модели и шаблоны, потраченные токены — для биллинга и контроля квот

2.3. Технические данные

  • IP-адрес — фиксируется в логах веб-сервера для безопасности
  • Тип и версия браузера, операционная система — для совместимости
  • События взаимодействия с интерфейсом (нажатия кнопок, открытие разделов) — для аналитики продукта в обезличенном виде

2.4. При оплате

Платежи обрабатываются сторонней системой ЮKassa (ООО НКО «ЮMoney»). Реквизиты вашей банковской карты в Сервис не передаются и нами не хранятся. Мы получаем только факт успешной/неуспешной оплаты и сумму.

3. Как мы используем данные

  • Для предоставления услуг Сервиса (обработка запросов, хранение истории чатов, экспорт)
  • Для авторизации и обеспечения безопасности учётной записи
  • Для биллинга и учёта израсходованных токенов
  • Для связи с вами по техническим вопросам и важным изменениям сервиса
  • Для улучшения качества сервиса (анонимизированный анализ поведения)
  • Для соблюдения требований законодательства РФ

4. Передача данных третьим лицам

Ваши данные могут передаваться следующим обработчикам:

  • Google LLC (Google AI / Gemini API) — тексты ваших запросов и приложенные файлы передаются для генерации ответов ИИ. Google обрабатывает данные согласно своей политике конфиденциальности.
  • ООО "РОБОКАССА" — данные платёжных операций для обработки оплаты.
  • Российские государственные органы — в случаях, прямо предусмотренных законодательством РФ.

Мы не продаём ваши данные и не передаём их в маркетинговых целях третьим лицам.

5. Хранение данных

  • Данные хранятся на серверах в Российской Федерации, что соответствует требованию ст. 18 152-ФЗ о локализации
  • Доступ к данным имеет только администратор сервиса
  • Пароли хранятся в виде необратимого хеша, восстановление исходного пароля невозможно
  • Срок хранения — пока существует ваша учётная запись. После удаления аккаунта данные удаляются в течение 30 дней
  • Аналитические события хранятся в обезличенном виде не более 90 дней

6. Cookies и localStorage

Сервис использует localStorage браузера для хранения токена авторизации (чтобы вы оставались залогинены). Cookies в традиционном виде сервис не использует, рекламные/трекинговые системы третьих лиц не подключены.

7. Ваши права

В соответствии с законодательством РФ вы имеете право:

  • Получить информацию о составе ваших персональных данных, которые мы храним

  • Запросить удаление вашей учётной записи и связанных с ней данных
  • Отозвать согласие на обработку персональных данных
  • Обжаловать действия Оператора в Роскомнадзор

Для реализации любого из этих прав отправьте запрос на support@conspectai.tech. Запрос будет рассмотрен в течение 30 дней.

Мы применяем разумные технические и организационные меры для защиты ваших данных:

  • Передача данных только по HTTPS (TLS 1.3)
  • Изоляция базы данных от внешней сети
  • Хранение паролей в виде bcrypt-хешей
  • Регулярные обновления зависимостей и безопасности

При этом ни одна система не может гарантировать абсолютной защиты — пожалуйста, используйте сложные уникальные пароли и никому не передавайте данные для входа.
